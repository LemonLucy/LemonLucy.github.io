summary: 

로봇 시스템
  이미지 수집: 이동형 로봇에 장착된 카메라가 작물의 이미지를 촬영
  이미지 처리: 촬영된 이미지는 딥러닝 모델 AI Hub의 노지 작물 해충 진단 이미지 API와 농진청의 작물 병해충 검색 서비스로 전송되어 병충해 여부를 판단
  모델: API는 병충해 여부를 분석하고 어떤 해충으로 인한 피해인지 결과를 반환 딥러닝 모델을 사용해 병충해 종류를 분류 및 진단하며 적합한 농약 살포 여부를 결정
  스프레이 모듈: 병충해가 감지된 위치에 자동으로 농약을 살포하는 장치
  통신 모듈: 로봇과 앱 간 데이터를 주고받는 역할을 하며 Wi-Fi 또는 Bluetooth를 사용
  센서: GPS 거리 센서 등 다양한 센서를 통해 작물의 위치와 병충해 상태를 감지

  딥러닝 모델 서버
  모델 배포: Python으로 학습된 모델을 TFLite로 변환해 로봇에 탑재하거나 서버에 배포하여 작동
  예측 처리: 로봇이 촬영한 이미지를 모델 서버로 전송해 병충해 여부를 분석하고 결과를 반환

  원격 조종 앱
  UI/UX: 사용자가 로봇을 제어하고 실시간으로 상태를 모니터링할 수 있는 사용자 인터페이스를 제공
  실시간 데이터 통신: 로봇의 상태 정보와 분석 결과를 받아 사용자가 확인
  제어 기능: 사용자가 로봇을 수동으로 이동시키거나 농약 살포 명령을 내릴 수 있는 기능을 포함
